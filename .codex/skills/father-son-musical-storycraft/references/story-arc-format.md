# docs/story-arc.md Format

`docs/story-arc.md` is the canonical mutable story artifact for this musical.

Use these sections:

1. `# Story Arc`
2. `## Premise`
3. `## Core Theatrical Device`
4. `## Character Arcs`
5. `## Lifecycle Structure`
6. `## Act Outline`
7. `## Scene And Song Map`
8. `## Motif Progression`
9. `## Open Creative Questions`
10. `## Revision Notes`

Update rules:

- Keep the file concise enough to read in one pass.
- Use headings and bullets for scanability.
- Preserve existing decisions unless the user requests a reset.
- Add a dated revision note for each meaningful update.
- Do not store full lyrics, full scenes, or score notation here unless explicitly requested.
